# 简介

起因：家里长辈手机不太行了，让我给她挑一个。
不过预算并不高，1300 以内
这个价格我本来就打算直接看中低端了，但是......
邦吧卡邦！天玑 8200 加入了队伍

# 为什么要选择这台手机

在逛京东的时候本来是打算看看中端和低端芯片的手机，比如骁龙 6 Gen 1,7s，天玑 1080，6080 啥的，后来无意中翻到了红米 Note12T Pro，然后想起来天玑 8100 和天玑 8200 这俩好东西，然后一搜，找到这么几款：

  1. iQOO Z8 8+256 软妹币 1399
  2. 红米 Note12T Pro 12+256 软妹币 1299
  3. iQOO Neo7 SE 12+256 软妹币 1599（好像是，记不清了

上面这些价格都是 1 月初（大概）的，是的当时 Neo7 SE 还没有这么实惠的价格
时间来到了除夕夜，她的手机又出现了点小问题，然后我寻思上京东看一下
此时价格：

  1. iQOO Z8 不变
  2. 红米 Note12T Pro 12+256 软妹币 1249
  3. iQOO Neo7 SE 12+256 软妹币 1399（银河颜色为 1400）

当时看到这个价格，u1s1，我们都觉得可以，但是当时还下定决心（毕竟过年或许会有点支出（？）
直到初三那天，我再次上京东看了一眼价格，1389（关注店铺入会领 10 元劵），当时家里人都聚在一块，她那破手机也实在是不太行，然后......
邦吧卡邦！支付成功
最终价格是 1377.92，银行卡支付再减 10 元，京东过年给了几块钱的红包用上了

背面 颜色：银河
正面

# 配置

iQOO Neo7 SE 搭载了天玑 8200 处理器，iQOO Z8、红米 Note12T Pro 同款
天玑 8200 可以视为天玑 8100 的升级版，而天玑 8100 无需多言，稍微了解点的都知道这是一颗神 U

# 屏幕

我手里这台使用的是天马的高频 PWM 调光 OLED 屏，相较于早期的国产、三星垃圾屏外加低频 PWM 调光，新批次的 iQOO Neo7 SE 使用的天马高频 PWM 则非常巴适，侧视不会出现发绿（点名批评 Xiaomi 12X），开启护眼模式后对眼睛很友好
相比之下 iQOO Z8 抽奖阴阳屏（新批次也存在）就很。。。而且 Z8 8+256 的价格就 1399 了，同样的价格 Neo7 SE 岂不更好？不管是从外观，还是从硬件方面来看，iQOO Neo7 SE 堪称千元王者

# 拍照

没啥可拍的，随便放两张
左 iQOO Neo7 SE 右 Xiaomi 12X




u1s1，vivo 拍天空、星空和人是真不错

# 优点

  1. 银河色真 tm 好看啊
  2. 拍照拍人和天空星空都不错，vivo 传统艺能属于是
  3. 性能妥妥的
  4. 续航不错，特别是待机续航
  5. 120W 满速充电嘎嘎爽

# 缺点

  1. 单扬声器效果一般般，勉勉强强吧，Camera&Music 或许不会太好
  2. 背面的摄像头模组有点大，需要忍一下（不是
    明明摄像头实际占地并不大缺给了那么大的空间，不过整体设计还是挺好看的
  3. 个人觉得机子挺大的但是或许还好？比 K40 大一个边吧，比 12X 大很多感觉，也不知道算优点还是缺点

# 总结

外观精美，手感可以，拍照够用，日用流畅
非常对得起 1389 这个价格👍👍👍
详细测评可参考：
【IT 之家评测室】iQOO Neo 7 SE 上手评测：首发天玑 8200，性能续航双向提升
【大家测】2099 元起售 iQOO Neo7 SE 开箱体验 | 首发天玑 8200 闪耀银河色调
不过这机子已经出了很久了，评测也失去了一些时效性
如果考虑屏幕和护眼的话，这边建议买 iQOO Neo7 SE，现在的 iQOO Z8 貌似容易买到阴阳屏，而且据说 Z8 那块 LCD 也一般，相比之下现在的 Neo7 SE 貌似都是天马高频 PWM 调光了，这块屏用起来蛮不错的也不累眼，显示效果更好
